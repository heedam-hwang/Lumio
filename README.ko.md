# Lumio

[English README](./README.en.md)

## 앱 소개

Lumio는 영어 책을 읽다가 모르는 문장이나 단어를 빠르게 기록하고 다시 복습할 수 있도록 설계된 앱입니다. 사용자는 카메라 또는 포토 라이브러리에서 페이지 이미지를 가져오고, 책별로 페이지를 정리한 뒤, OCR로 추출된 문장을 확인하고 필요한 단어를 단어장에 저장할 수 있습니다. 최근에는 스크린샷 없이도 전용 `단어 검색` 탭에서 단어를 바로 검색하고 최근 조회를 관리할 수 있도록 확장되었습니다.

## 주요 기능

- 카메라 또는 포토 라이브러리에서 영어 책 페이지를 촬영하거나 불러올 수 있습니다.
- 업로드한 페이지 이미지에 OCR을 적용해 텍스트를 추출하고 문장 단위로 분리합니다.
- 문장과 단어를 열어 한국어 번역 결과를 확인할 수 있습니다.
- `단어 검색` 탭에서 단어를 직접 입력해 뜻을 확인하고 최근 조회 목록을 관리할 수 있습니다.
- 영어 문장과 단어의 발음을 TTS로 들을 수 있습니다.
- 중요한 단어를 개인 단어장에 저장할 수 있습니다.
- 현재 조회 결과, 최근 조회, 단어장에서 같은 뜻 수정 UI로 번역 결과를 직접 보정할 수 있습니다.
- 페이지를 책 단위로 정리하고 저장된 페이지 순서를 관리할 수 있습니다.

## 기술 개요

- SwiftUI 기반 iOS 앱
- SwiftData를 사용한 로컬 데이터 저장
- Vision OCR을 사용한 영어 텍스트 감지
- NaturalLanguage를 사용한 문장 분리
- Translation 프레임워크를 사용한 영어-한국어 번역
- AVSpeechSynthesizer를 사용한 영어 발음 재생

## 현재 구조

- `홈`: 책 목록과 업로드 진입점
- `단어 검색`: 직접 단어 검색, 최근 조회, 웹 사전 이동, 뜻 수정
- `책 상세`: 선택한 책의 페이지 목록과 페이지 편집
- `페이지 상세`: OCR 결과 문장 확인, 번역, 단어 선택
- `단어장`: 저장한 단어 복습

## 스크린샷

- `Word Search`: 스크린샷 추가 예정
- `Recent Lookups`: 스크린샷 추가 예정
- `Meaning Edit`: 스크린샷 추가 예정
